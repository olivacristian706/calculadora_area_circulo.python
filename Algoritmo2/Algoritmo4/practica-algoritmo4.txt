------------------------------------------------------------------------------------
Problemática 3:
El Usuario quiere saber si puede entrar a una discoteca según la edad del cliente
------------------------------------------------------------------------------------

------------------------------------------------------------------------------------
Algoritmo:
1-Solicitar al usuario que ingrese la edad del cliente
2-Verificar si la edad ingresada cumple con el requisito para ingresar a la discoteca
3-Mostrar al usuario si su cliente puede o no ingrasar a la discoteca
------------------------------------------------------------------------------------

------------------------------------------------------------------------------------
Pseudocódigo:

Inicio
    # Paso 1: Solicitar al usuario que ingrese la edad del cliente
    Mostrar mensaje:"Por favor, ingrese la edad del cliente: "
    Leer el dato ingresado y asignarlo a variable edad

    # Paso2: Verificar si la edad ingresada cumple con el requisito para ingresar a la discoteca
    Si edad >= 18 entonces
        asignarle a la variable permitido igual a verdadero
    sino
         asignarle a la variable permitido igual a falso
    Fin si

    # Paso3: Mostrar al usuario si su cliente puede o no ingrasar a la discoteca
    Si permitido es verdadero
        Mostrar mensaje: " Puedes ingresar a la discoteca!"
    sino    
        Mostrar mensaje: " Lo sentimos mucho, No puedes ingresar a la discoteca siendo menor de edad"
    Fin si
Fin
